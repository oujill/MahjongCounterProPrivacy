最後更新日期：2025/10/15

麻將日誌 服務（以下簡稱「麻將日誌」、「本服務」和「本公司」）非常重視您的線上個人隱私。為了讓您瞭解我們將如何蒐集、應用及保護您所提供的資料，請詳閱下列隱私權保護聲明。

隱私政策

蒐集之資料類別

  • 裝置基本資訊（例如作業系統、App 版本）
  • 使用情況統計（例如遊玩時間、操作紀錄）
  • 若使用「雲端同步」功能，會同步儲存使用者遊戲資料至雲端

本服務不會蒐集精確的裝置定位資訊或敏感個人資料。

資料的運用

  • 提供麻將日誌功能
  • 改善 App 使用體驗
  • 統計分析使用情況

廣告與第三方服務

本 App 使用第三方廣告服務（例如 AdMob）來顯示廣告。這些服務可能會蒐集匿名的使用資訊，如裝置識別碼、使用習慣及 App 使用情況，以提供更精準的廣告。 這些資料僅用於廣告或分析用途，不會用於識別個人。如不希望被收集，可關閉 App 或使用裝置的廣告追蹤限制功能。

雲端同步與使用者權利

若使用者啟用「雲端同步」功能，本服務會將遊戲資料儲存至雲端。 若您希望刪除雲端資料，或對資料使用有疑問，請聯絡客服：mahmahsup@gmail.com

安全性

服務提供者會盡力保障使用者資料安全，採取物理、電子及程序性措施保護資料。

其他資料使用

本服務可能會收集匿名化、彙整的使用統計資料，用於改善 App 功能與使用體驗，這些資料不會用於識別個人，也不會提供給第三方。

關於 Cache / 本地儲存

本 App 可能使用本地儲存（如 Cache）來提升使用效能及同步資料，所有資料皆僅用於提供服務功能。
